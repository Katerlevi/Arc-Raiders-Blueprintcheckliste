# Arc-Raiders-Blueprintcheckliste
Klickt einfach die Blueprints an, die ihr gefunden habt. Ihr könnt die Internetadresse direkt als Browserquelle in OBS einfügen um Eurem Chat den Fortschritt sichtbar zu machen.
Danach einfach in der Browserquelle in OBS rechtsklick und auf "Interagieren" klicken. Dann öffnet sich ein Fenster. Darin könnt ihr dann die Blueprints anklicken. 

Viel Spaß damit!
